Architecture Overview

  • Hosted on a private S3 bucket (static website hosting disabled)
  • Secured with CloudFront (CDN) + Origin Access Control (OAC)
  • Served over HTTPS via AWS Certificate Manager (ACM)
  • Custom domain registered with Amazon Route 53
  • Automated deployments using GitHub Actions for CI/CD

When I push code to GitHub, an automated GitHub Actions workflow defined in a deploy.yml file triggers a CI/CD pipeline. It syncs the updated files to my private S3 bucket and invalidates the CloudFront cache to reflect the changes. The site is delivered globally via CloudFront (CDN), secured with HTTPS using ACM, and DNS resolution is handled by Route 53.
