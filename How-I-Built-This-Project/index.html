Architecture Overview

  • Hosted on a private S3 bucket (static website hosting disabled)
  • Secured with CloudFront + Origin Access Control (OAC)
  • Served over HTTPS via AWS Certificate Manager (ACM)
  • Custom domain registered with Amazon Route 53
  • Automated deployments using GitHub Actions for CI/CD

When I push code to GitHub, a GitHub Actions workflow automatically builds and syncs the site to S3. CloudFront securely delivers the site over HTTPS using an ACM certificate, with Route 53 resolving DNS queries to the CloudFront distribution.
